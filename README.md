## 📖 Overview

The TANWAR GYM Management System is built with HTML, CSS, and JavaScript
to manage gym members, calculate BMI, search records, and track
memberships using Local Storage.

## 🚀 Getting Started

### Prerequisites

-   Visual Studio Code
-   Web Browser

### Installation

``` bash
git clone https://github.com/yourusername/gym-management-system.git
cd gym-management-system
```

### Run

Open `index.html`.

## 💻 Technologies Used

-   HTML5
-   CSS3
-   JavaScript (ES6)
-   Local Storage API

## ⚙️ How It Works

1.  Add member.
2.  BMI calculated.
3.  Save to Local Storage.
4.  Search/Delete members.
5.  Dashboard updates.
